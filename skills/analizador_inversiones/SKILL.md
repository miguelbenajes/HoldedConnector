# Analizador de Inversiones (Skill de Ejemplo)

Esta skill permite a Antigravity (tu asistente) realizar análisis financieros profundos utilizando herramientas personalizadas o scripts que tú definas.

## Instrucciones
1. Siempre revisa la tabla `invoices` para calcular el flujo de caja.
2. Utiliza la lógica de margen de beneficio definida en el proyecto.
3. Si detectas una anomalía en los datos, notifica inmediatamente al usuario.

## Recursos
- Scripts de ayuda en `scripts/` (opcional).
- Ejemplos de prompts en `examples/` (opcional).
